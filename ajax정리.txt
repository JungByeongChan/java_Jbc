화면 JSON =>
		=< JSON 서버

특징
1. ajax로 본ㄹ떄 contentsType을 json으로 설정
2. ajax로 서버에서 보낸 데이터를 받을 때 dataType을 json으로 설정
3. 데이터를 보낼 때 JSON형태로 변환해서 전송
4. 서버에서는 map에 담아서 전송
5. 서버에서 화면으로 보낼 데이터를 VO에 받을 때 @RequestBody를 이용하여 받음

-화면에서 보낸 데이터를 서버에서 VO에 담고, 서버에서는 여러 데이터를 한번에 보낼 때
$.ajax({
	data : JSON.stringify(sendObject), // 객체를 json으로 변환해서 전송
	contentType : "application/json; charset=UTF-8", // 화면에서 json으로 보냄
	dataType : "json" // 서버에서 json으로 받음
});
